# Projekt LAN - Cisco Packet Tracer

## Popis sítě
Síť se skládá ze dvou segmentů propojených switchi S1 a S2. 
Obsahuje DHCP/DNS server pro správu adres a domén, a Web server s jednoduchou strankou.
